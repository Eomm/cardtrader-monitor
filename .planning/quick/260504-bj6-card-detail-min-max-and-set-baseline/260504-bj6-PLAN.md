## Task 2: Add "Set current as Baseline" button below the price chart

**Files:** `src/pages/CardDetailPage.tsx`

**Action:**
- Add state: `const [settingBaseline, setSettingBaseline] = useState(false)`
- Add handler `handleSetBaseline`:
  ```ts
  async function handleSetBaseline() {
    if (!card || card.latest_price_cents === null) return;
    setSettingBaseline(true);
    const { error: updateError } = await supabase
      .from('monitored_cards')
      .update({ baseline_price_cents: card.latest_price_cents })
      .eq('id', card.id);
    setSettingBaseline(false);
    if (!updateError) {
      setCard((prev) => prev ? { ...prev, baseline_price_cents: prev.latest_price_cents } : prev);
    }
  }
  ```
- Inside the Price History block (the `priceHistory.length > 1` conditional, after `<PriceChart ...>`), add the button:
  ```tsx
  <div className="mt-3 flex justify-end">
    <button
      type="button"
      onClick={handleSetBaseline}
      disabled={settingBaseline || card.latest_price_cents === null}
      className="rounded-md bg-slate-700 px-3 py-1.5 text-xs font-medium text-slate-300 transition-colors hover:bg-slate-600 disabled:opacity-50"
    >
      {settingBaseline ? 'Saving...' : 'Set current as Baseline'}
    </button>
  </div>
  ```

**Verify:**
- Button appears below the chart when price history exists
- Clicking updates the Baseline display in the price row
- Button is disabled while saving

**Done:** Button renders below chart and updates baseline correctly
