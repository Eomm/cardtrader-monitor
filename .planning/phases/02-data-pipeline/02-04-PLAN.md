         supabase.from('price_snapshots').select('price_cents').eq('monitored_card_id', card.id).order('recorded_at', { ascending: false }).limit(1)

       - If error: show error message

       - If cards.length > 0: show ImportWishlistForm in compact mode (compact=true) at top, then CardList below
